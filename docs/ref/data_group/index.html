Data

Classes for serializing and storing data.

Classes

class ByteBuffer Simple wrapper for a refcounted contiguous array of bytes, analogous to NSData on iOS or byte[] in Java.
class ByteBufferStream A `Stream` that acts on a `ByteBuffer`.
class FileStream A `Stream` backed by a `FILE*`.
class LocalStore Extremely simple data storage concept.
class Stream Abstract base class for a thing that can read and write elementary data.
